# Ain'tWatchingAllat: Summarize YouTube Videos

To start summarizing YouTube videos with Ain'tWatchingAllat, follow these steps:

## Installation
Clone the frontend repository from GitHub.

## Run
```bash 
npm run dev
```

## View in the Browser: Open http://localhost:3000 with your browser to access the app.

PS: the BACKEND is closed source and all calls are rate limited, the LLM models i used are opensource but fine tuned by me. if there is an error contact me because the API is probably unactive/the RENDER instance is IDLE
